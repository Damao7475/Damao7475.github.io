## 博客框架 Hexo
Hexo是一个快速、简洁且高效的博客框架，支持Markdown以及各种强大插件，使用和部署见[官网](https://hexo.io)，这里使用的是挂载到github page上。
### 配置
在_config.yml中修改对应参数，包括网站布局信息（`site`），网站网址（`URL`），写作格式（`Writing`），部署信息（`deploy`）

## 博客主题 Next
Next是一个开源的Hexo主题，简洁优雅，支持各种插件，安装和使用见[这](https://github.com/theme-next/hexo-theme-next)。
### 主题配置
#### 设置menu
修改相关格式设置，在主题`_config`配置文件中找到`menu`设置自己所需导航栏的名称和图标。

输入命令：
```
hexo new page <name> #<name>替换为你要加入的导航栏名称
```
实际上会在/source下新建/name文件夹。

#### 用户头像
在主题的`images`文件夹中放入你的图片,并在主题`_config`配置文件中找到`avator`填写路径即可

## 功能添加
### 搜索
```
npm install hexo-generator-searchdb
```
在主题`_config`配置文件中找到`local_search`并设置`enable=true`
在根目录`_config`配置文件中添加：
```
# hexo-generator-searchdb
search:
  path: search.xml
  field: post
  format: html
  limit: 10
```

### 添加阅读时间字数提示
```
npm install hexo-symbols-count-time
```
在主题`_config`配置文件中找到`symbol_count_time`并设置对应项为`true`
在根目录`_config`配置文件中添加：
```
# hexo-symbols-count-time
symbols_count_time:
  symbols: true
  time: true
  total_symbols: true
  total_time: true
  exclude_codeblock: false
  awl: 2    
  wpm: 275
  suffix: "mins."
```

### 添加不蒜子统计
可以统计观看人数和观看时间
在主题`_config`配置文件中找到`busuanzi_count`并设置`enable=true`
